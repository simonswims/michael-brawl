# Michael's Brawl Stadium

A mobile-friendly game combining football, Brawl Stars-style combat, and chicken nuggets!

## Controls
- **Move**: Drag the joystick (bottom left)
- **Shoot**: Tap the football button (bottom right)
- **Power-ups**: Collect 🍗 and 🍟 for special abilities
